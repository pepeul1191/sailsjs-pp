# demo-sails

Instalar dependencias:

    $ npm install

Arrancar la aplicación:

    $ sails lift

Configuración de la ruta de los arhicos estáticos esté en el archivo:


    .sailsrc

Configuracines adicionales en fuente[4]

Generar un hook:

    $ sails generate hook <<nombre_hoook>>
